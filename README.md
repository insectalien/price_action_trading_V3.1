🚀 台股 PA 智能雷達與量化回測系統 (V3.1 Elite)
這是一個進化版的 Python 量化交易系統。除了核心的價格行為 (Price Action) 策略分析外，本版本新增了全市場即時雷達，能自動從台股數千檔標的中篩選出資金最集中的熱門股，並自動完成策略回測與資料庫存檔。

🌟 本次版本更新 (V3.1)
智能市場雷達 (Market Radar)：自動掃描全台股成交金額前 N 名標的，捕捉市場熱點。

SQL 數據持久化：整合 PostSQL 資料庫，自動記錄每場回測的勝率、盈虧比與詳細交易日誌。

健壯性優化：針對 FinMind API 進行深度封裝，具備自動登入、異常捕獲與非交易時段保底機制。

多執行緒架構潛力：模組化設計，支持從單一標的回測擴張至全市場掃描。

🛠️ 核心功能模組
Scanner (市場探測器)：利用 tick_snapshot 技術，即時定位資金流向，過濾權證與 ETF，精準鎖定熱門個股。

Strategy (PA 策略內核)：專精於「看漲/看跌吞沒」形態，並輔以 MA20 趨勢濾網與量能確認。

Backtester (專業回測引擎)：支援動態停損 (Stop Loss)、停利 (Take Profit) 以及「反向訊號即時離場」邏輯。

Data Center (數據中樞)：整合 PostgreSQL 高效存儲架構，實現金融數據自動化抓取、清洗與本地化持久化，極大化策略回測與掃描之讀取效率。。

📊 實測表現 (以熱門標的為例)
系統具備自動參數優化功能，在標準實驗中表現卓越：

策略核心：看漲吞沒 + 趨勢過濾

風險控制：固定百分比停損 (預設 10%)

歷史勝率：最高可達 80.00% (依標的特性有所不同)

存檔機制：整合 PostgreSQL 關聯式資料庫，自動存儲每場回測之勝率、盈虧比與詳細交易日誌，支援高效 SQL 交叉檢索與大數據分析。
